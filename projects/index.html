Cadcam Invaders | 2018

Developed with Aaron Atkinson

This project was an assignment for COSC 190 - Intermediate Programming. The program was coded in Java and JavaFX. The assignment was to create a UFO invaders game like the classic 'Space Invaders'. Why 'Cadcam Invaders'? Well you could say that the Cadcam program, is the CST nemesis; so obviously we must defend the planet against the evil Cadcam and their 3D printed UFOs.

After entering a name and clicking play we have gameplay that starts off relatively easy but can quickly get out of hand. There are ten levels within the game, as you level up the following changes are applied to the game:

  • Increase number of UFOs
  • Increase speed if UFOs
  • Increase reload time.

Completing the game is difficult but still possible to achieve. If you do manage to win you are taken to the high score screen; otherwise when a UFO reaches the bottom of the screen the player losses and the game over screen is displayed with the original Doom death animation.

Once the game is over, you are taken to a high score screen showing the top three players and their high score. From here you can save and exit from the game.
